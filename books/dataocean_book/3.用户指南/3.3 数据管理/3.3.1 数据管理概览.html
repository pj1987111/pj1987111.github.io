3.3.1 数据管理概览

数据管理概览

数据管理概览主要是对一些指标做一些统计。
指标包括存储相关，数据质量分，访问量等等。

3.3.1.1 存储相关

存储相关的概览包括存储增长趋势图，项目存储，个人存储以及表存储。

3.3.1.2 数据质量得分

根据规则模板计算每个用户的数据质量得分。

3.3.1.3 访问量相关

访问量相关的指标包括3个月未访问的表统计，30天访问量统计。 点击表名可以跳转到数据查找界面。
