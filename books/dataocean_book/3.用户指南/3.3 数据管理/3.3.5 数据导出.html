3.3.5 数据导出

数据导出顾名思义就是支持平台的数据导出成csv或者execl等格式的后下载到本地。

注意事项
* 导出的数据量较大时优先使用export命令导出数据，导出速度较快,使用详情见产品帮助手册SQL语法。
* 导出数据为select查询结果，要求sql语句必须为select sql语句，一次只能导出一条select语句，导出文件限制200万条
* 例如：select partnercode, count(1) as req_count from databaseName.tableName where ds='20170621'
