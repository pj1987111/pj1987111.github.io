docker命令

Thu, Oct 10, 2019

  1. docker中 启动所有的容器命令

    docker start $(docker ps -a | awk '{ print $1}' | tail -n +2)

  2. docker中 关闭所有的容器命令

    docker stop $(docker ps -a | awk '{ print $1}' | tail -n +2)

  3. docker中 删除所有的容器命令

    docker rm $(docker ps -a | awk '{ print $1}' | tail -n +2)

  4. docker中 删除所有的镜像

    docker rmi $(docker images | awk '{print $3}' |tail -n +2)

  5. 复制文件到镜像中

    docker cp src $dockerid:dest

  6. docker执行命令

    docker exec $dockerid mv /usr/share/openstack-dashboard/static/dashboard/css/6d0969e4da42.css /usr/share/openstack-dashboard/static/dashboard/css/6d0969e4da42.css_bak
